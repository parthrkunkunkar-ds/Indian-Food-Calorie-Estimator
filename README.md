# 🍛 Indian Food Calorie Estimator

An AI-powered desktop application that detects Indian food from images 
and estimates calories using deep learning.

## Features
- 📸 Image upload with preview
- 🧠 MobileNetV2 Transfer Learning (80 classes, 61% accuracy)
- 🔥 Calorie + macro estimation (Protein, Carbs, Fat, Fiber)
- 📊 Top 3 predictions with bar chart
- 🔎 Manual food name lookup
- ⚖️ Adjustable portion size slider
- 📅 Daily calorie goal progress bar

## Setup
pip install -r requirements.txt
python App.py

## Dataset
Indian Food Images Dataset — 3957 images, 80 classes
